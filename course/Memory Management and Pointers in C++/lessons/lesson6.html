Memory Management and Pointers in C++

Master the essentials of memory management and pointers with engaging lessons and exercises.

Introduction

Welcome to the lesson on Memory Management and Pointers in C++. This lesson will cover key concepts such as stack vs. heap memory, pointers, dynamic memory allocation, smart pointers, and best practices to avoid memory leaks.

Lesson Content

Understand the differences between stack and heap memory, and how they are used in C++.

Learn the basics of pointers, how to declare them, and how to use them to access memory addresses.

Exercises

Fundamental

Write a program that declares a pointer to an integer, assigns it to the address of a variable, and then displays the value stored at that address using the pointer.

Create a program that performs pointer arithmetic to iterate through an array of integers and prints each value.

Implement a program that dynamically allocates an array of integers using the `new` keyword and then frees the memory using `delete`.

Write a program that demonstrates the use of a pointer to a function, passing different functions to the pointer and calling them.

Intermediate

Create a program using double pointers (pointers to pointers) to dynamically allocate a 2D array.

Develop a program that dynamically allocates memory for a 2D array and allows the user to input values.

Write a program that creates a class Person with a name attribute. Use a unique_ptr to create an instance of the Person class, assign a name, and display it.

Create a program that simulates a shared resource using shared_ptr . Create two shared_ptr instances pointing to the same dynamically allocated memory and display the reference count.

Advanced

Implement a class with a constructor and destructor where memory is dynamically allocated in the constructor and freed in the destructor. Intentionally create a scenario that causes a memory leak, then fix it.

Create two classes A and B , where each has a shared_ptr to the other. Identify the issue of circular reference and resolve it using weak_ptr .

Implement a custom deleter for a unique_ptr that dynamically allocates an array of integers. The custom deleter should correctly free the allocated memory.

Write a simple memory pool allocator that preallocates a large block of memory and manages allocation and deallocation within that block. Demonstrate the usage of this custom memory pool with a few allocations and deallocations.
