Basic Exercises

Exercise 1: Simple Lambda Expression

Write a lambda function that takes two integers and returns their sum.

Exercise 2: Capturing Variables in Lambdas

Create a C++ lambda function that captures a local variable by reference and modifies its value.

Exercise 3: Smart Pointer Basics (Unique Pointer)

Use `std::unique_ptr` to manage a dynamically allocated integer.

Exercise 4: Move Semantics and Rvalue References

Demonstrate move semantics using a class with dynamic memory.

Intermediate Exercises

Exercise 5: Lambda with Return Type

Write a lambda expression that takes two floats and returns the larger one.

Exercise 6: Smart Pointer (Shared Pointer)

Use `std::shared_ptr` to manage memory and share ownership of an object.

Exercise 7: Move Constructor vs Copy Constructor

Demonstrate the difference between move and copy constructors.

Exercise 8: constexpr Functions

Write a `constexpr` function that calculates the power of a number.

Advanced Exercises

Exercise 9: Generic Lambda with Smart Pointers

Write a generic lambda that works with both `std::unique_ptr` and `std::shared_ptr`.

Exercise 10: Implementing a Move-Only Class

Create a class that can only be moved, not copied.

Exercise 11: constexpr with Classes

Write a C++ class with a `constexpr` constructor and use it at compile time.

Exercise 12: Lambda Expressions and Function Templates

Write a templated function that accepts a lambda expression and applies it to an array.
