Multithreading Exercises

Basic Exercise 1

Create a simple thread that prints "Hello, World!" to demonstrate basic thread creation.

Basic Exercise 2

Use two threads to increment a shared counter. Synchronize access using a mutex to prevent race conditions.

Basic Exercise 3

Write a program with three threads that print numbers in sequence: Thread 1 prints 1-5, Thread 2 prints 6-10, Thread 3 prints 11-15.

Basic Exercise 4

Implement a program where one thread writes data to a file while another reads from it simultaneously.

Intermediate Exercise 1

Develop a multithreaded program to find the sum of an array using multiple threads. Each thread computes a partial sum.

Intermediate Exercise 2

Simulate a producer-consumer scenario using threads. Use a mutex and condition variables to handle synchronization.

Intermediate Exercise 3

Create a thread pool with fixed-size threads that process a queue of tasks. Implement task submission and execution.

Intermediate Exercise 4

Implement the dining philosophers problem using threads and semaphores to manage resource sharing and avoid deadlocks.

Advanced Exercise 1

Implement parallel matrix multiplication using multiple threads, dividing the task into row-wise computations.

Advanced Exercise 2

Develop a multithreaded HTTP server simulation where each request is handled by a separate thread from a thread pool.

Advanced Exercise 3

Create a parallel quicksort algorithm that uses multiple threads to sort sections of the array concurrently.

Advanced Exercise 4

Design a thread-safe logging system where multiple threads write log entries to a shared file without data corruption.
